# Wendy Logan | Portfolio

This is my personal portfolio website built with HTML, CSS, and JavaScript. It displays my GitHub projects dynamically using the GitHub API.

## Features
- Profile photo with clean circular crop
- Dynamic loading of public GitHub repositories
- Clickable project cards
- GitHub-themed dark color palette
- Responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- GitHub API
